# 05 Methodological Addenda and Controls

This directory contains supplemental methodological notes that clarify design intent and comparison boundaries for the frozen study. These notes are provided for transparency and reader alignment only.

Scope constraints
- No new experiments, prompts, variables, or evaluation dimensions are introduced.
- Nothing in this directory is used to compute, select, or modify any reported result.
- Where any wording here diverges from the frozen protocol or released artifacts, the frozen protocol and artifacts take precedence.

## Role in the supplement

Materials here serve three limited purposes.
- Explain design choices that are referenced but not expanded in the main text.
- Make the intended separation between exploratory probing and protocolized measurement explicit.
- State what comparisons are and are not supported by the released setup, to reduce over-interpretation.
